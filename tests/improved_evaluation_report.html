🔍 Technical Insights & Conclusion

The Zero-Recall Problem is Solved: In the baseline implementation, the model achieved a high aggregate accuracy (95.59%) solely by predicting the majority class (Normal), resulting in a Balanced Accuracy of exactly 50.00%. The model completely failed to identify the minority class (Delayed Myelination).

Impact of Interventions:

  • Data Augmentation (MRI): Simulating physiological artifacts (Gibbs ringing, bias fields) successfully upsampled the minority class, providing the fusion transformer with 10,000 perfectly balanced records.
  • Grey Wolf Optimizer (GWO): Replaced grid-search heuristics by probabilistically converging on an optimal configuration space for the Late Fusion Transformer, specifically dialing in the learning rate, dropout matrices, and modality-attention boundaries.

Final Assessment: The pipeline is now clinically viable for inference. The extreme discrepancy between Macro F1 and Overall Accuracy has been completely normalized.
